# Portfolio Website

A personal portfolio website built with **vanilla JavaScript** — no frameworks, just clean fundamentals.

## Features

- Responsive layouts using media queries, optimised for desktop and mobile
- Smooth scrolling and interactive UI animations
- Modular, clean-code structure for easy maintainability

## Tech Stack

`HTML` `CSS` `JavaScript`

## Getting Started

```bash
git clone https://github.com/your-username/portfolio.git
cd portfolio
open index.html
```

No dependencies. No build step. Just open in a browser.
